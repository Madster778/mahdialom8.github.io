Bootcamp 2020: Ball Dodge Game

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project uses concepts such as arrays, loops, and randomness. I created a project in which balls are spawned from a class designed to hold information only to do with the balls (e.g. position, color and size). The balls are then spawned from outside of the sketch, in random positions, where they will begin to travel towards the centre of the sketch. The user will then have a different ball that follows the mouse which you use to dodge the balls that will be bouncing around the sketch. However, the collision does not work at this current moment in time and will need to be corrected at a later time as the game does not function as intended.
